# 观察者模式

### 设计原则

> 为了交互对象之间的松耦合设计而努力。

### 设计模式

> 观察者模式:定义了对象之间的**一对多**依赖，这样一来，当一个对象（Subject）改变状态时，他的所有依赖（Observer）着都会受到通知并自动更新。

```
Subject是真正拥有数据的人，Observer是主题的依赖者，只有一个对象控制这份数据。
```
